# Makhafola-Khafo Trading & Projects

A professional, modern, and fully responsive corporate website for **Makhafola-Khafo Trading & Projects**. 

Founded in 2013, Makhafola-Khafo is a South African company offering premium services in the construction, stationery supply, and cleaning industries. This project is built using semantic HTML5, pure CSS3 variables and layout systems, and vanilla JavaScript.

---

## 🚀 Key Highlights & Design Aesthetics

*   **Elite Typography**: Styled using a premium pairing of *Playfair Display* (serif, for elegant headings) and *Source Sans 3* (sans-serif, for crisp body text).
*   **Harmonious Color Palette**: Built on a curated corporate blue palette (`#1A4FAF`, `#143D8A`, `#EEF3FB`, `#D0DEFA`) for a premium and trustworthy visual appearance.
*   **Fluid & Dynamic Layouts**: Fully responsive design leveraging CSS Grid and Flexbox to ensure perfect display on desktops, tablets, and mobile devices.
*   **Micro-Animations**: Features smooth fade-in animations triggered by viewport entry via the vanilla JavaScript `IntersectionObserver` API.

---

## 🛠️ Technology Stack

*   **Structure**: HTML5 (Semantic elements)
*   **Styling**: Vanilla CSS3 (Custom properties/variables, modern grid layouts, custom transition animations)
*   **Interactivity**: Vanilla JavaScript (Modern ES6+, high-performance `IntersectionObserver`)
*   **Typography**: Google Fonts integration (Playfair Display & Source Sans 3)

## 💻 Local Development & Deployment

To run and preview the website locally on your machine:

1.  **Clone the Repository**:
    ```bash
    git clone git@github.com:Kayfalls/Makhafola-Khafo.git
    cd Makhafola-Khafo
    ```
2.  **Open the HTML File**:
    Simply double-click the `makhafola_khafo_website.html` file or open it directly in any modern web browser:
    *   Chrome / Chromium
    *   Firefox
    *   Safari
    *   Edge
3.  *(Optional)* **Using a Dev Server**:
    If you are using Visual Studio Code, you can install the **Live Server** extension and click **"Go Live"** to run it on a local development port with hot-reload features. Or run using Python's built-in server:
    ```bash
    python3 -m http.server 8000
    ```
    Then visit `http://localhost:8000/makhafola_khafo_website.html` in your browser.
